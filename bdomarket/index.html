    get_market_price_info(ids, sids, convertdate=True, formatprice=False) async ¶

    Get price history for an item or list of items. If multiple ids are given, returns a JsonArray of JsonObject of the items' price history. If only one id is given, returns a JsonObject of the item's price history.

    Parameters:

    Name Type Description Default
    ids List[str]

    itemid(s)

    		 required
    sids List[str]

    subid(s) like enhancement level

    		 required
    convertdate bool

    Convert unix-like timestamp to UTC datetime. Defaults to True.

    		 True
    formatprice bool

    Format price, adding separator (,). Defaults to False.

    		 False

    Returns:

    Name Type Description
    ApiResponse ApiResponse

    standardized response. Returned values in content.history: key (eg. "1745193600000"): Unix timestamps in milliseconds (from utils use ConvertTimestamp), value (eg. 75000000000): item silver value
